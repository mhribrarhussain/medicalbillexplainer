# 30-Day US SEO Traffic Growth Plan

## Phase 1: Launch & Indexing (Days 1-5)
- **Day 1: Deploy to Netlify**
    - Drag and drop the `Medical Bill Explainer` folder to Netlify Drop.
    - Set custom domain (if purchased).
- **Day 2: Search Console Setup**
    - Create Google Search Console account.
    - Submit `sitemap.xml`.
    - Use "URL Inspection" tool to manually request indexing for `index.html` and 5 key CPT pages (e.g., 99213, 80053).
- **Day 3: Social Signals**
    - Share the "Medical Bill Explainer Tool" on Reddit communities like r/HealthInsurance, r/PersonalFinance, and r/PovertyFinance.
    - **Note:** Do not spam. Offer it as a helpful free resource.
- **Day 4: Performance Check**
    - Run PageSpeed Insights on Homepage and one CPT page.
    - Aim for Green (90+) score on Mobile.
- **Day 5: Bing Webmaster Tools**
    - Submit sitemap to Bing Webmaster Tools (often faster indexing).

## Phase 3: Traffic Analysis & Optimization (Days 16-30)
- **Day 20: Check Search Console**
    - Look at "Performance" tab.
    - Identify keywords with high impressions but low clicks.
    - Update Meta Titles of those pages to be more click-worthy (e.g., add "Cost" or "2024 Price").
- **Day 25: New Content**
    - If "MRI" pages are ranking, add more specific MRI codes.
    - If "EOB Guide" is ranking, add a "How to Appeal a Denial" guide.
- **Day 30: Schema Validation**
    - Use Google Rich Results Test to verify FAQ Schema is appearing for CPT pages.

## KPI Targets (Day 30)
- **Indexed Pages**: 50+
- **Organic Clicks**: 10-50/day
- **Avg Position**: Top 20 for long-tail queries like "CPT 99213 cost".
